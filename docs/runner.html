Runner:

Mini jeu avec une interface graphique.

Le joueur contrôle un vaisseau spatial voguant dans l'espace.

Il doit éviter les obstacles qui lui tombe dessus et aller le plus loin possible.

Un vaisseau ennemi apparait!

Préparez vos cannons pour pouvoir l'affronter et continuer votre route.
